public class AggregationProtocolObserver
extends Object
implements Control

Observes an AggregationProtocol during the simulation, prints the results to the standard output or to a file for plotting, and determines when to stop the simulation.

This control expects the following parameters in the configuration file:

protocol - the name of protocol to observe.
precision - stop the execution when the variance of all estimates is less than or equal to precision.
plotfile - (optional) name of a file to write the output to . If this parameter is set, the output will be formatted for Gnuplot.
